The Publisher Module¶

The Publisher is a module made to publish your web application. It comprises of two panels: on the left, the maps list and, on the right, the configuration tabs.

Maps List¶

If you click on a map name from the list, the Publisher module will reload using the map you have choosen and giving the capability to change the configuration and publish again.

Configuration tabs¶

The Options which you can set in the Publisher interface are grouped by tabs, you should click on one of the following icons, to access the parameters.

Warning

Every modification made to the any tabs requires to press the Publish button to save your modifications.

Icon Tab Action
General General configuration
Layers Layers configuration
Map Map Configuration
Control Tools Configuration

As Per default, in the activated tab, the settings you can made are listed below.

The Description field is set using the WISIWYG editor seen previously, its value will be used to define the abstract metadata information.

Layout configuration¶

By choosing the layout, you decide the way elements will be displayed in the final application. You are invited to first choose the Fullscreen option.

Layers configuration¶

Base layers¶

The base layers are optional, you can display only one proprietary base layer per project. Once you select one proprietary option (Google, Bing or IGN) you have access to the list of available base layers.

By using the Default option, you can choose amongst the activated base layers the one to use as per default.

Note that you can use your own published project as base layers. Indeed once a MapMint project is published, the setup of mapcache was automatically made with all the layers defined as activated by dafault in the project.

Note

You can use *Ctrl*+Click to select multiple base layers from a list.

Overlays configuration¶

You are invited to set all layers as Raster and as Activated.

You are now invited to activate the popup option for the new_poi_names_wake layer and set it as Vector.

Tools Configuration¶

In this section you can activate tools which will be available in the published application.

You are invited to select the following options:

  • Pan
  • Zoom Box
  • Zoom to max extent
  • Profile tool
  • Mouse position
  • Search Box

Select None for Spatial Analysis Tools.

Note

Use Ctrl + Click to add multiple tools at the same time.

Publish an application¶

Publish and preview¶

Once you have made all the settings required, you can publish application by clicking on the Publish button on top of the right panel. Then, you can preview your project by clicking on the Preview button.

Access the map publicly¶

To access the map publicly, use the link MapMint Client on your Desktop or directly use the following link: http:localhost/mapmint/ui/public/Sample1.

If you hover mouse on top of a green points, you should see the popup window you created in the previous section.

To use the Profile tool, first click on the Profile button, then draw a line on the map, once your line is finished, double click on the end position and you will see the elevation profile displayed.

To use the search engine, type few letters in the text field and wait for the list of available featured displayed, then click on one in the list to zoom on it.
